# 【Pythonライブラリのインストール】

ライブラリのインストール方法や今回の開発で必要なライブラリを紹介

## ライブラリのインストールコマンド
```
pip install [ライブラリ名]
```

## Djnagoのライブラリをインストール
```
pip install django
```

## 統計および機械学習で必要なライブラリをインストール
```
pip install numpy
pip install pandas
pip install scikit-learn
pip install ultralytics
pip install japanize-matplotlib 
pip install jupyter

「pip install matplotlib」 はultralyticsでインストールされる為不要
```

## インストール済みライブラリの一覧表示
```
pip list
```
