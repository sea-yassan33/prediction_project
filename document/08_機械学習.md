# 【機械学習とPython】

## 機械学習を行うプロセス
![Alt text](./img/img13.jpg)

```
データを分析するプロジェクトでは、データの準備、データ加工、前処理、データの可視化・分析、モデリング、検証、運用という流れで進める。
```
